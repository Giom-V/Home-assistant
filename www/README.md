# WWW

This folder is used to store static files that can be served to the Home
Assistant frontend. Any files placed in this directory can be accessed via the
`/local/` URL.

Most of the content in this folder is installed and managed through the
[Home Assistant Community Store (HACS)](<https://hacs.xyz/>).

**Note:** Files in this folder are publicly accessible if your Home Assistant
instance is exposed to the internet.

**Documentation:** <https://www.home-assistant.io/docs/frontend/serving-files/>
